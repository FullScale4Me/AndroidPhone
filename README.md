Accessing your phone via USB cable

Connect to your PC using the USB cable.

Check the notifications on your phone by swiping the screen from the top to bottom.

If the connection has been recognized by your phone you should see a two line notice:

'USB for charging phone'
'Tap for other USB options.'

Tap this notification.

In the section named 'Use USB for' select 'Transferring files / Android Auto'

Enter your PIN when prompted and tap 'OK'.

Your phone should now appear as:

'SAMSUNG Android'

in the File Manager section 'Devices'.
